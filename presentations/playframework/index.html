What is Play Framework ?

  • A pure Java framework
  • A clean alternative to the bloated JEE frameworks
  • A source of immense productivity for Java programmers
  • A reason for Java programmers to finally feel joyful

Why should I use Play?

  • Has all the benefits of working in Java
  • Large ecosystem of libraries
  • All the benefits of the JVM
  • Favorite tools and IDE's

Creating a new Play Project

	    play new path_to_project

If you plan to use Eclipse

	    play eclipsify path_to_project

Running the application

	    play run path_to_project

View the application

Point your browser to http://localhost:9000 and you will see the default page

Where did that come from ?

	    /app/views/Application/index.html

Let's display a simple Pomodoro image on the landing page

  • Copy the image to /public/images/

What's happening under the hoods ?

All web applications have a common layout

  • In Play a view template can inherit from a super-template
  • All view templates which inherit from the same super template will share the same layout

Looks pretty lame... Where do I put my stylesheets ?

            /public/stylesheets

Hey! We do not want to block the thread...

  • Play's default threadpool only has n+1 threads for managing requests
  • But Play has a powerful functional programming library with support for Continuations
  • Play also supports Non Blocking Web Requests with the WS library

Can we show an activity indicator while the Pomodoro is doing it's stuff ?

  • Play comes bundled with jquery 1.5.2 min
  • We can use it to control the visibility of an animated gif to show an ajax loader

Can I track my Pomodoros ?

Sure! Play comes bundled with a persistence layer

  • Play uses JPA, and comes bundled with a default implementation using Hibernate
  • We need to create a Pomodoro Entity
  • We can use the CRUD module to view the data
  • The CRUD module is highly customizable and we can change the UI to confirm with the rest of our application
